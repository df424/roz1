# P1 Audio Subsystem -- Wiring Diagram

Connection diagram for the P1 head audio subsystem. See [design.md](design.md) for
specifications and rationale.

## Parts

| Ref | Part | Connectors |
|-----|------|------------|
| J1  | Jetson Orin Nano | 4x USB 3.2 Type-A |
| D1  | Cubilux USB-A DAC (192KHz/24-bit) | USB-A male, 3.5mm TRRS female |
| A1  | Adafruit MAX98306 Class D Amp | Screw terminals: L+/L-/R+/R-, VDD, GND, SHDN, 3.5mm input header |
| S1  | 40mm 4 Ohm Speaker (Adafruit 3968) | Solder pads (+/-) |
| U1  | Plugable USB Audio Adapter | USB-A male, 3.5mm mic jack, 3.5mm headphone jack |
| M1  | Electret Mic Capsule - Left Ear (Adafruit 1064) | Solder pads (signal/ground) |
| M2  | Electret Mic Capsule - Right Ear (Adafruit 1064) | Solder pads (signal/ground) |
| C1  | STM32G071RB Controller | GPIO output (SHDN control) |
| PWR | P1 5V Servo Rail | +5V / GND |

## Cable Summary

| Cable | From | To | Type | Length |
|-------|------|----|------|--------|
| USB (speaker) | J1 USB-A port 1 | D1 USB-A male | Direct plug | -- |
| Analog audio | D1 3.5mm out | A1 3.5mm input | 3.5mm pigtail or patch cable | ~10 cm |
| Speaker wire | A1 L+/L- terminals | S1 +/- pads | 22 AWG 2-conductor | ~5-10 cm |
| USB (mic) | J1 USB-A port 2 | U1 USB-A male | Direct plug | -- |
| Mic cable | U1 3.5mm mic jack | M1 (or M1+M2) | Shielded 2-conductor + 3.5mm plug | ~15 cm |
| Amp power | PWR +5V/GND | A1 VDD/GND | 22 AWG 2-conductor | ~10 cm |
| Mute control | C1 GPIO + GND | A1 SHDN + GND | Single wire + ground | ~10 cm |

## USB Port Usage

| Jetson USB-A Port | Device |
|-------------------|--------|
| Port 1 | D1 -- Cubilux USB DAC (speaker output) |
| Port 2 | U1 -- Plugable USB Audio (mic input) |
| Port 3 | Available (dev keyboard, second mic adapter, etc.) |
| Port 4 | Available (debug adapter, etc.) |
